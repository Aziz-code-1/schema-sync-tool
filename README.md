# sql-schema-comparison

## Описание
Проект для сравнения структуры таблиц между двумя схемами PostgreSQL.

## Цель проекта
Выявление различий в структуре таблиц между schema_a и schema_b для контроля качества данных (schema drift detection).

## Функциональность
- Поиск пересекающихся таблиц между схемами
- Сравнение колонок таблиц
- Анализ различий в:
  - типах данных
  - длине полей
  - NULL / NOT NULL ограничениях
- Вывод только различий

## Используемые технологии
- PostgreSQL
- information_schema
- SQL (JOIN, CTE, CASE WHEN, INTERSECT)

## Бизнес-логика
Используется для:
- контроля структуры DWH
- проверки изменений схем данных
- подготовки данных для аналитики
- выявления несоответствий между источниками

## Как запустить
1. Создать schema_a и schema_b
2. Запустить SQL скрипт compare_schemas.sql
3. Получить таблицу различий
